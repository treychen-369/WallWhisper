# WallWhisper 效果短视频拍摄脚本

## 视频一：30 秒效果片（社交媒体传播用）

### 定位
- 平台：B站、小红书、Twitter/X、微信朋友圈
- 目标：让观众在 30 秒内理解 "摄像头自动说英语" 的效果
- 风格：真实家庭场景，不做作，有温度

### 分镜脚本

| # | 时长 | 画面 | 音频/字幕 | 备注 |
|---|------|------|----------|------|
| 1 | 0-3s | 客厅全景，安静，能看到墙角/天花板上的萤石 C7 摄像头 | 🔇 安静 + 字幕：`"家里的摄像头，能自动教英语？"` | 建立悬念 |
| 2 | 3-6s | 孩子从画面外走进客厅（自然走动，不要刻意表演） | 🔇 脚步声 | 关键：让摄像头能拍到孩子走过 |
| 3 | 6-8s | 摄像头特写，指示灯亮起（可以是萤石 App 的告警截图叠加） | 字幕：`"人形检测触发 ✓"` | 可用手机录屏萤石 App 告警提示 |
| 4 | 8-18s | 回到客厅画面，摄像头喇叭传出 Emily 的声音 | 🔊 Emily: `"Hello! Cat! Can you say cat? Meow!"` + 字幕（中英双语） | **核心镜头！** 一定要能清晰听到摄像头出声 |
| 5 | 18-23s | 孩子的反应（可能咯咯笑、尝试说 cat、或者好奇地看摄像头） | 🔊 孩子的自然反应 | 真实反应比完美表现更打动人 |
| 6 | 23-28s | 孩子继续玩耍，Emily 中文解释传来 | 🔊 Emily: `"Cat是小猫咪！你能说cat吗？喵！"` | 展示中英双语 |
| 7 | 28-30s | 结束画面 + Logo/项目名 | 字幕：`"WallWhisper - 墙上的低语者"` + `"GitHub: WallWhisper"` | 可配轻快 BGM 渐弱 |

### 拍摄建议

📸 **设备**：
- 手机即可（横屏 16:9）
- 固定在三脚架或支在桌上，不要手持晃动

🎬 **拍摄技巧**：
- 提前确认 Emily 在运行中，让孩子自然走进客厅
- 不要告诉孩子 "走过去"，让她自然出现
- 如果第一次没拍好，可以等冷却时间后再试
- 关键是要录到摄像头喇叭出声的瞬间
- 手机放在能同时拍到孩子和摄像头的位置

🔊 **音频注意**：
- 摄像头喇叭音量要调到中等偏大
- 录制环境要安静（关掉电视、手机静音）
- 如果摄像头喇叭音量不够，可以后期稍微调高

✂️ **后期**：
- 保持真实，不要过度剪辑
- 加字幕（Emily 说的英语 + 中文翻译）
- 可以加一个简短的开头字幕说明场景

## 视频二：3 分钟技术解说片（B站/YouTube 长视频）

### 定位
- 平台：B站、YouTube、知乎
- 目标：让技术人/家长了解原理并想要自己搭建
- 风格：技术解说 + 真实效果 + 部署演示

### 分镜脚本

| # | 时长 | 画面 | 旁白/字幕 | 备注 |
|---|------|------|----------|------|
| **Part 1: Hook（抓注意力）** |
| 1 | 0-5s | 效果片的精华镜头（孩子走过 → Emily 说英语） | `"家里的摄像头，不仅能看家，还能教英语"` | 先放效果，再解释 |
| 2 | 5-15s | 你面对镜头（可以不出镜，用屏幕录制+旁白代替） | `"我是一个程序员爸爸。我用家里已有的萤石摄像头和小米路由器，花了零硬件成本，给女儿搭了一个 AI 英语外教。"` | 建立人设：技术人+爸爸 |
| **Part 2: 原理讲解** |
| 3 | 15-35s | 架构图（用 README 中的 Mermaid 图，或用 Figma/PPT 做动画版） | `"原理很简单：摄像头检测到人 → 路由器上的程序收到告警 → 调用 AI 生成英语内容 → 语音合成 → 推回摄像头喇叭播放。整个过程 10 秒内完成。"` | 配合架构图中的箭头逐步出现 |
| 4 | 35-55s | OpenClaw 界面截图 / Emily SOUL.md 内容滚动 | `"Emily 的人设是用 OpenClaw 定义的。她知道我女儿叫什么、几岁、喜欢什么。她会记住教过的每个单词，避免重复。"` | 展示 OpenClaw 的价值 |
| 5 | 55-70s | RTSP Backchannel 技术图解 | `"最酷的一点是，声音是直接推到摄像头喇叭的。RTSP Backchannel 反向音频通道，不需要额外的蓝牙音箱。"` | 可以画一个简单的信号流动画 |
| **Part 3: 效果演示** |
| 6 | 70-100s | 拼接 3 个不同场景的真实效果：① 早上起床 Emily 说 good morning ② 下午放学回家 Emily 打招呼 ③ 晚上睡前 Emily 讲 bedtime story | `"早上，Emily 会说早安。下午放学回来，她会教一个新单词。晚上睡觉前，她会讲一个简单的英语小故事。"` | 展示三种模式 |
| 7 | 100-120s | 多轮对话演示（如果有录到的话）：Emily 说 → 孩子回应 → Emily 接着说 | `"如果孩子回应了，Emily 还能接着对话，最多进行 5 轮。"` | 可选。如果没有好的录像就跳过 |
| **Part 4: 快速部署** |
| 8 | 120-140s | 终端录屏：`git clone` → `pip install` → `config.yaml` 编辑 → `python run.py test_tts` | `"部署非常简单。克隆项目，安装依赖，填配置，就能跑。"` | 用 asciinema 或终端录屏，加速播放 |
| 9 | 140-155s | Docker 部署录屏 / 路由器 SSH 截图 | `"如果想 24 小时运行，用 Docker 部署到路由器或者 NAS 上。我家用的是小米 7000 路由器。"` | 展示 Docker + 路由器的极客感 |
| **Part 5: 号召行动** |
| 10 | 155-170s | GitHub 仓库页面 / Star 按钮 | `"项目完全开源，MIT 协议。如果你家也有萤石摄像头，不妨试试。链接在简介和评论区。"` | 引导 Star 和尝试 |
| 11 | 170-180s | 回到孩子玩耍的画面，Emily 在背景中说英语 | `"让英语学习像呼吸一样自然。"` + 项目 Logo | 温馨结尾 |

### 拍摄素材清单

需要准备的素材：

1. **效果录像**（最重要！建议多拍几次取最好的）
   - [ ] 孩子自然走进客厅，Emily 自动说英语（30 秒效果片的素材也可以复用）
   - [ ] 早上 Emily 说 good morning 的录像
   - [ ] 下午/傍晚 Emily 打招呼的录像
   - [ ] 睡前 Emily 讲故事的录像（可选）
   - [ ] 孩子回应 Emily 的对话（可选，但有就加分）

2. **技术录屏**
   - [ ] 终端安装部署过程（可以加速）
   - [ ] 萤石 App 告警截图（手机录屏）
   - [ ] OpenClaw 界面截图（如果有的话）

3. **图片素材**
   - [ ] 家里摄像头的实拍照片
   - [ ] 路由器照片（如果是小米路由器更好）
   - [ ] 架构图（从 README 的 Mermaid 图导出 PNG）

### 后期建议

🎵 **BGM**：
- 推荐：轻快的吉他/ukulele 背景音乐
- 音量控制在旁白的 15-20%
- 推荐来源：YouTube Audio Library（免费商用）

📝 **字幕**：
- 全程加中文字幕
- Emily 说英语时加中英双语字幕
- 技术术语首次出现时加括号注释

🎨 **风格**：
- 不要过于专业严肃，保持"程序员爸爸分享"的亲切感
- 孩子的真实反应 > 完美的表演
- 技术部分简洁清晰，不要深入代码细节
